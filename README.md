# Raise a Criminal

Raise a Criminal is a command line game where the player raises a child from age 1 to 18. Throughout the game, the player is presented with different situations and must decide how to respond as a parent. These choices affect different aspects of the child's development, such as parental neglect, school engagement, peer influence, household stability, substance exposure, impulsivity, and mental health.

The idea behind the project came from a simple question: can a machine learning model be combined with a choice based game to show how small decisions and life circumstances can gradually shape a person's future?

As the child grows up, the player's decisions accumulate into a profile. Some choices also create different life paths. For example, deciding whether the child attends school changes which situations appear later in the game. Similarly, peer groups and mental health support can affect future events and opportunities.

The machine learning part of the project was built using a K-Nearest Neighbors (KNN) classifier. Before training the model, several variables were grouped into broader risk factors. Instead of using dozens of individual columns, related factors were combined into features such as abuse, parental neglect, school neglect, substance exposure, household instability, peer delinquency, impulsivity, and mental health.

I chose KNN because it fits the idea of the project well. Rather than creating complex rules, KNN compares the final child profile to similar examples it has seen before. In simple terms, the model asks: "Among people with similar levels of neglect, instability, school problems, and peer influences, what outcomes were most common?".

After training, the model and scaler were saved using Joblib so that the game could load them instantly without retraining every time it runs. During gameplay, the player's choices update the feature values, and at the end of the game those values are scaled and passed into the trained model.

Once the child reaches age 18, the game generates a final risk score and displays the factors that contributed most strongly to that outcome. The result is not intended to predict real criminal behaviour. Instead, it serves as an educational demonstration of how machine learning can be integrated into an interactive application and how multiple risk factors can accumulate over time.

This project was built using Python, Pandas, Scikit-learn, and Joblib. The entire game runs in the terminal and combines machine learning, branching narratives, and feature engineering into a single application.

### Disclaimer

This project is for educational purposes only. The predictions generated by the model should not be interpreted as real-world assessments of individuals. Human behaviour is influenced by many factors that cannot be fully captured by a dataset or machine learning model.
